## Responsive Behavior

### Desktop (1024px+)
- 3-column grids for articles
- 2-column layout (main + sidebar) for details
- Full navigation bar
- Large hero sections

### Tablet (768px - 1023px)
- 2-column grids
- Stacked content on detail pages
- Collapsed navigation
- Medium hero sections

### Mobile (<768px)
- Single column layout
- Hamburger menu
- Stacked filters
- Compact cards
- Small hero sections
